# Legacy Index

This repository is used by the pre-FreeCAD v1.0 Addon Manager to find available Addons, and is retained here so old versions of FreeCAD can still install these packages. New versions of FreeCAD's Addon Manager source their index from the new [Addon Index][New]. To request addition to the index please see the documentation in the [Addon Academy].

[New]: https://github.com/FreeCAD/Addons
[Addon Academy]: https://freecad.github.io/Addon-Academy/Guides/Publishing/Indexed
